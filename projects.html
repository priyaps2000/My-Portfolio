Gen AI Chatbot for ESG Benachmarking Automation
(MAR 2024)

This project was developed as part of a hackathon, with the goal of automating the manual process of ESG (Environmental, Social, and Governance) benchmarking using Generative AI. Our solution aimed to automate collection of metrics such as carbon footprint, diversity, and inclusion etc. from ESG reports of companies and trusted websites providing ESG ratings.

We designed a conversational AI chatbot capable of interacting with users to gather input PDFs and generate detailed ESG reports. The chatbot was powered by a Retrieval-Augmented Generation (RAG) framework, leveraging OpenAI GPT-3.5, LangChain models, Chroma DB and FastAPI. We utilized Selenium and Beautiful Soup for web scraping to retrieve ESG metrics from websites. Additionally, we developed a feature to generate user-friendly, formatted PDF reports summarizing ESG metrics. These reports were securely stored and shared via Azure Blob Storage. To provide a seamless user experience, we built a responsive web interface using Chainlit.The solution was hosted in Azure cloud.

We won first place in the hackathon by delivering a fully functional prototype within two days, showcasing the ability to rapidly build and deploy an impactful solution.

Efficient Activity Recognition Using Wi-Fi CSI         (AUG 2021-MAY 2022)

Have you ever thought that a Broadcom WiFi chip could recognize what you are doing inside a room using WiFi signals from your home router?

Yes! It doesn't require wearables, sensors, or cameras. We apply machine learning techniques on channel state information(CSI) from WiFi signals to identify human gestures.
